Каталог игр:

Веб-приложение "Сеть по продаже ключей для компьютерных игр" для покупки игровых ключей steam.
Использует API IGDB для получения данных об играх.

Функции пользователя с ролью «Гость»:
- регистрация и аутентификация;
- поиск, фильтрация и сортировка игр;
- просмотр комментариев и рейтинга игр;
Функции пользователя с ролью «Пользователь»:
- покупка игр;
- изменение и удаление аккаунта;
- оценка товаров;
- добавление и удаление в избранное;
- добавление и удаление в корзину;
- добавление, изменение и удаление комментариев;
- просмотр истории посещенных игр;
- общение с продавцом через сообщения;
Функции пользователя с ролью «Администратор»:
- управление каталогом игр;
- управление пользователями и продавцами;
Функции пользователя с ролью «Продавец»:
- изменение и удаление аккаунта;
- продажа игр;
- редактирование цены игр;
- взаимодействие с покупателями;


Технологии:

- Frontend: React 19.2.0, TypeScript 5.9.3, Vite 7.2.2
- Backend: Node.js 22.0, Express.js 5.1.0
- База данных: mssql 12.2.0 с Microsoft SQL Server 20
- Прокси: nginx 1.29.5
- API: IGDB (Twitch), Stripe API, Google OAuth 2.0 
- Контейнер: Docker 4.68.0 с docker-compose 2.3.0
